# Fungal Ecology

Fungal dispersal, connectivity, and ecological spread.

## Connected Projects

- [Long Read Sequencing for Arbuscular Mycorrhizal Fungi](https://cu-esiil.github.io/AMFungi_Biogeography_Diversity/): Using long-read sequencing to synthesize arbuscular mycorrhizal fungi biogeography and diversity.
- [BioViewPoint](https://cu-esiil.github.io/BioViewPoint/): Visualization-focused project space for ecological interpretation and biological insight.
- [Fungal Dispersal](https://cu-esiil.github.io/fungal_dispersal/): Open synthesis on fungal dispersal, connectivity, and ecological spread.
