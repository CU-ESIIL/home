# Style Guide Panels

Store future thumbnail reference panels here.

Use this folder for visual guide images that should influence a full thumbnail-generation pass, such as contact sheets, motif studies, or approved aesthetic direction panels.

The current user-provided concept direction is a pure-white background with highly dimensional, layered, paper-like or relief-like forms. If that direction is adopted for a future full set, store the guide panel here and reference it in the generation prompt for that set.

Suggested practice:

- keep the original guide image filename intact when possible
- add a short note in the generation prompt or manifest describing which set it informed
- if a guide introduces a new house style, update the repository image guidance before making that set active
