# Ecosystem Directory

The homepage is intentionally curated. This directory is the deeper index for
exploring the wider OASIS ecosystem across working groups, projects, events,
libraries, and templates.

### Cohort 1

- [BioViewPoint](https://cu-esiil.github.io/BioViewPoint/) — Visualization tools for biodiversity data.
- [Extreme Wildfire](https://cu-esiil.github.io/ExtremeWildfire/) — Investigating extreme wildfire behavior.
- [Fungal Dispersal](https://cu-esiil.github.io/fungal_dispersal/) — ESIIL working group on fungal dispersal.
- [Macrophenology](https://cu-esiil.github.io/macrophenology/) — Macroecological patterns in phenology.
- [Maka-Sitomniya](https://cu-esiil.github.io/Maka-Sitomniya/) — Research on traditional ecological knowledge.
- [AI for Natural Methane](https://cu-esiil.github.io/AI-for-Natural-Methane/) — Harmonizing natural methane datasets using AI.
